# Redux Book List

Really basic React app with a list component ...

## Getting Started

```shell
> git clone https://github.com/CodeSchwert/redux-book-list.git
> cd redux-book-list
> npm install
> npm start
```

Browse to `http://localhost:8080` to view the app.
